geodrawer

The perfect library to use in the admin area of a modern web application in order to provide geolocalization info to all kind of entities.
Highly customizable, responsive and written focusing on the ease of use, can be easily integrated inside every project.

No css or static assets import, only a single js file (around 60 kb).

Download

Download/clone the repository, or use bower if you prefer:

$ bower install geodrawer
